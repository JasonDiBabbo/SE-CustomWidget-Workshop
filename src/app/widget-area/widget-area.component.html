Custom Widget Area

This is where the custom widget that you're designing will be placed while testing. Below are the steps for testing your custom widget.

  1. Place the content of widget.html into widget-area.component.html.
  2. Place the content of widget.css into widget-area.component.css.
  3. Insert the functional components of widget.js into widget-area.component.ts.

Note: The widget.json file isn't accounted for in this environment. If you have any widget fields, hard code their values for testing.

Emulating Events

The buttons on the left side of the screen can be used to emulate stream events similar to how the StreamElements overlay interface provides.

Note: This environment is not meant to be a replacement for the StreamElements overlay design interface. Use the event emulate to ensure baseline functionality and then migrate your widget to the StreamElements interface for further testing.
